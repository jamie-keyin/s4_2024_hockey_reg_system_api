# Hockey Registration System API

This project is a Spring Boot REST API for managing hockey games and teams.

## Features
- Add, update, delete, and view games
- Connects to a MySQL database
- Swagger UI available for testing endpoints

## Getting Started
1. Ensure MySQL is running and a database named `hockey_db_2024` exists.
2. Update your `application.properties` file with correct DB credentials.
3. Run the app:
   ```bash
   mvn spring-boot:run

Once the app is running, visit:

http://localhost:8080/swagger-ui/index.html
